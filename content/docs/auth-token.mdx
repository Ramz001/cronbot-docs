## Steps to get token

1. Open Discord in browser.
2. Press `F12` → Open Developer Tools.
3. Go to `Network` tab.
4. Send message in any chat.
5. Search `messages` request.
6. Check `Headers` → `Authorization`.
7. Copy token value.

Keep token secret! ⊥ Share with nobody.

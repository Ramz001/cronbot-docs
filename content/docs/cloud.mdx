# Cloud Service

Use hosted service → fast setup. Zero infrastructure needed. Currently free (hosted on personal VPS). SaaS experience.

## How to use

1. Go to [Cronbot Dashboard](#).
2. Input Discord Auth Token (see [Auth Token Guide](/docs/auth-token)).
3. Set cron schedule.
4. Save & Deploy.

That's it. Task runs on our servers.
